
`du -d 1 -b /apps/○○○○/`  
`du`  
ディスク使用量を表示するコマンド。  
`-d 1`  
深さ1のディレクトリを表示します。  
指定したディレクトリの直下にあるサブディレクトリのサイズを表示します。  
`-b`  
バイト単位でサイズを表示します。通常は人間が読みやすい形式（KB, MBなど）で表示されますが、  
このオプションを指定することで、サイズをバイト単位で表示します。
例）
```
237793687       /apps/○○○○/x-batch
129451003       /apps/○○○○/y-batch
180273901       /apps/○○○○/z-batch
```
